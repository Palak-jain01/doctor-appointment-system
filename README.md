# Advanced Doctor Appointment Booking System

## Features
- React.js Frontend
- Doctor Listing
- Appointment Booking
- Login Page
- Appointment History
- Responsive UI
- Local Storage Integration
- React Router Navigation

## Run Project

1. Open terminal

2. Install dependencies:
npm install

3. Run project:
npm start

4. Open browser:
http://localhost:3000

## Skills Demonstrated
- React.js
- Components
- Hooks
- Routing
- Form Handling
- Local Storage
- Responsive Design
